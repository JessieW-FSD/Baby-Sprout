# Baby Sprout

A simple, all-in-one baby care tracking app for iOS. Built with SwiftUI and SwiftData.

## Features

### Tracking

- **Feeding** — Log bottle feeds (mL) and breastfeeding sessions (duration per side, first side)
- **Sleep** — Track sleep sessions with start/end times and Live Activity on the lock screen
- **Diapers** — Record wet, poo, and combined changes with amount tracking
- **Supplements** — Log vitamins and medications with dosage
- **Growth** — Track weight (kg), height (cm), and head circumference over time
- **Custom Events** — Capture milestones and moments with photos, titles, and descriptions

### Dashboard

The home screen shows a real-time summary of your baby's day:

- Age display (months and days)
- Feeding stats (session count, total mL, breastfeeding minutes)
- Sleep summary (total and average duration)
- Diaper counts with time since last change
- Latest growth measurements
- Quick-add buttons for all categories

### Live Activity & Widget

- Lock screen Live Activity shows elapsed sleep time when baby is sleeping
- Dynamic Island support with a "Wake" button to end the session
- Home screen widget showing current sleep status

### Data Export

Export your tracking data as **CSV** or **PDF** for sharing with your pediatrician or keeping records.

### Other

- Onboarding flow to set baby's name and date of birth
- Configurable day start hour for overnight tracking
- Optional 4-hour reminders
- App group container for widget data sharing

## Requirements

- iOS 18.0+
- Xcode 26.0+

## Getting Started

1. Clone the repository
2. Open `BabySprout.xcodeproj` in Xcode
3. Build and run on a device or simulator
